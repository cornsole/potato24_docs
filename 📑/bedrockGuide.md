### 2. 접속 방법

1. 마인크래프트 베드락 에디션(BE)을 실행합니다.
2. 메인 화면에서 **‘플레이’** 탭을 클릭합니다.
3. 화면 상단의 **‘서버’** 버튼을눌러 **‘서버 추가’** 버튼을 누릅니다.
4. 아래 정보를 입력합니다:
   * 서버 이름: 원하는 이름 (예: 감자서버)
   * 서버 주소: **be.potato24.kr**
   * 포트: **19132**
5. **저장** 후 생성된 서버 목록에서 감자서버를 선택해 접속합니다.
